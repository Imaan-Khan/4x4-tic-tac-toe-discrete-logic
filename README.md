# 4x4-tic-tac-toe-discrete-logic
Aims and Objectives
The point of this project is to design a 4×4 Tic Tac Toe game using basic digital logic ICs. Our project focuses on applying concepts learned in Digital Logic Design, such as combinational logic, sequential circuits, logic gates, decoders, and flip-flops.
Objectives of the project include:
•	To design a functional 4×4 Tic Tac Toe game using standard logic ICs
•	To understand game logic implementation using combinational and sequential circuits
•	To enhance practical knowledge of IC interfacing and hardware design
•	To simulate and verify the design using Proteus software	
•	To develop teamwork, problem-solving, and logical thinking skills
Project Description
The aim of tic tac toe is for each player to mark the grid, for example player 1’s mark is an “X”( or a Yellow led) and player 2’s mark is an “O” (or a red LED), each player takes turn to place their marks on the grid and the objective is to align 4 similar marks either vertically, horizontally or diagonally in order to win.
Our game of 4x4 tic tac toe is implemented entirely using digital logic ICs. Turn by turn players 1 and 2 selects a position on the 4×4 grid using input switches. The system ensures turn-based play by storing each move, and checks for winning conditions using logic circuits. Red- and yellow-colored LEDs are used to display each player’s moves and indicate the winner. A reset button has also been implemented to completely restart the game.
Claims and Achievements
This project successfully demonstrates the practical implementation of a digital game using only hardware components. The key achievements include:
•	Successful implementation of a 4×4 Tic Tac Toe game without microcontrollers
•	Accurate turn control between Player 1 (Red LED) and Player 2 (Yellow LED)
•	Correctly identifying the winning conditions and draw conditions
•	Functional Indication of a player 1 win, player 2 win or a draw
•	Verification through Proteus simulation before hardware implementation
